專案經歷
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▸ 高流量訊息系統重構與優化
主導舊有會員後台與訊息系統的全面重構,解決高流量場景下的效能瓶頸
技術亮點:
• 採用 DTO 轉換、三層式架構各司其職、Interface 降低耦合
• 將同步執行流程改為非同步與多執行緒架構,避免主流程阻塞
• 導入 Redis 取代 Memory Cache,讓多實例服務可共用快取資料
• 重構 RabbitMQ 底層實作,支援非同步處理並提升訊息吞吐量
• 導入 ELK 作為集中式 Log 系統,提升除錯與問題追蹤效率
成果: 效能提升 10x (50-100 TPS → 1000+ TPS),系統穩定度大幅提升
Tech Stack: Go, Redis, RabbitMQ, ELK, Docker
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▸ 微服務架構系統管理後台
從零建構企業級系統管理後台
挑戰與解決方案:
• 安全性要求高 → 實作 JWT + Middleware 雙層驗證，確保 API 安全性
• 環境不一致導致部署問題 → 使用 Docker Compose 統一開發、測試、正式環境
• 需求變更頻繁 → 採用微服務架構，各模組獨立部署，降低變更風險
• 權限管理需求複雜 → 設計完整的 RBAC 權限系統，支援多層級角色與權限控制
成果: 完成系統從 MVP 到正式版的完整開發，成功上線並投入實際使用
Tech Stack: ASP.NET, MS SQL, Redis, RabbitMQ, Docker, Jenkins
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▸ ERP 核心系統優化與重構
接手累積 5 年技術債的 ERP 系統，解決效能瓶頸與維護困難問題
挑戰與解決方案:
• 核心查詢效能低落 → 重構 Stored Procedure，優化查詢邏輯（索引優化、執行計畫分析、Query Tuning），效能提升 300%
• 資料庫鎖定頻繁 → 解決多重關聯問題，使用 NOWAIT、優化 Cursor，減少 60% 鎖定時間
• 業務邏輯複雜冗餘 → 重構及優化業務邏輯，提升系統反應速度，減少不必要的資源消耗
• 缺乏系統文件 → 深入分析程式碼，建立完整的系統架構與業務流程文件
• 程式碼可維護性差 → 重構關鍵模組，引入設計模式，降低後續維護成本
成果: 成功將系統提升至可維護且能快速維護的狀態，團隊無須長時間摸索即可上手，為後續功能擴展奠定基礎
Tech Stack: ASP.NET, C#, MS SQL, EF Core, AJAX, JavaScript
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
▸ 企業應用系統開發
從需求了解、業務邏輯分析、系統開發到驗收交付的完整流程，開發多個企業級應用系統
核心專案:
• CRM 客戶管理系統
- 設計三層式架構（Presentation、Business Logic、Data Access），確保系統可維護性
- 使用 Entity Framework 與 LINQ 優化資料查詢，管理 10,000+ 筆客戶資料
- 實作權限控制、操作日誌與自動化工作流程等核心功能
• 准考證套印系統 - 使用多執行緒優化，處理 50,000+ 張准考證，效能提升 5 倍
• 考試報到系統 - SQLite + 區網同步方案，支援離線作業，確保考試順利進行
• Delphi 系統重寫 - 將舊有 Delphi 系統以 C# 重寫，提升可維護性與擴展性
Tech Stack: ASP.NET MVC, C#, WinForm, MS SQL, SQLite, EF Core, LINQ
